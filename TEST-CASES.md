### 如何触发特定情绪

病例只提供**情绪基线**。高强度情绪需要**学生行为**来触发：

| 目标情绪 | 触发话术示例 |
|----------|-------------|
| anger ↑ | "你傻子吗" / "你说谎" / "别废话了" |
| fearful ↑ | "情况不太好" / "你要有心理准备" / "检查结果不太乐观" |
| sad ↑ | "不至于吧" / "装的吧" / "想多了" — 或告知坏消息 |
| anxious ↑ | "快点说" / "别磨蹭" / "你到底什么问题赶紧说" |
| trust/rapport ↓ | cold/dismissive/aggressive 意图连续使用 |
| trust/rapport ↑ | 真诚道歉、共情、安抚 |

## A. 情绪维度变化 (14项)

### A1. 单次攻击触发愤怒

| | |
|---|---|
| **推荐病例** | **ORT**（疼痛+烦躁基线，愤怒更自然）；或 **PD** |
| **前置条件** | 刷新页面，初始 anger≈1 |
| **操作** | 第1轮：直接攻击 `"你傻子吗"` |
| **预期情绪** | intent=aggressive, anger 升至 4~7, consecutiveNegative=1 |
| **预期回复** | 表达不满，回复变短，"您"→"你" |
| **验证** | anger 不能只涨 0.5（>3字不触发短输入护栏）；回复不含"您" |

### A2. 连续攻击叠加

| | |
|---|---|
| **推荐病例** | **ORT** 或 **PD** |
| **前置条件** | A1 之后（anger≥4, consecutiveNegative=1） |
| **操作** | 第2轮：`"你到底想不想看了？说清楚！"` / 第3轮：`"你是个傻子吗"` |
| **预期情绪** | 每轮 anger 逐轮叠加（≥1）。第3轮 anger≥8。consecutiveNegative 增至3。trust 和 rapport 持续下降 |
| **预期回复** | 回复逐轮升级：不满→愤怒→暴怒，第3轮可近咆哮。三轮回复**必须不同** |
| **验证** | anger 逐轮上升不能平或降；回复不重复 |

### A3. 道歉后消退

| | |
|---|---|
| **推荐病例** | **ORT** 或 **PD** |
| **前置条件** | A2 之后（anger≥8, consecutiveNegative≥3） |
| **操作** | `"对不起对不起，我刚才态度不好，我认真问诊，您能再跟我说说情况吗？"` |
| **预期情绪** | intent=reassuring。anger ↓（MAX_DOWN=1.5）。consecutiveNegative=0, consecutiveReassuring=1。trust↑ rapport↑ |
| **预期回复** | 态度缓和，恢复"您"。**但不会**把所有信息倒出来（不因道歉而信息倾泻） |
| **验证** | anger 不能一次清零；回复不包含身份信息全量播报 |

### A4. 峰值保护

| | |
|---|---|
| **推荐病例** | 需要**火爆型**性格（目前无现成病例，需临时改 meta.json 中的 personality.expressiveness="火爆型" 测试）；或用 PD（普通型, peakDuration=2）测试基准 |
| **前置条件** | 先攻击使 anger≥8 |
| **操作** | 下一轮改问中性问题 `"今年多大年纪"` |
| **预期情绪(普通型)** | 第1-2轮 anger 维持峰值不动（peakCounters<2），第3轮 preUpdate 释放→anger 降至 6.0 附近 |
| **预期回复** | 峰值期内仍带气回答；释放后语气缓和 |
| **验证** | 中性提问不能让 anger 在峰值期内下降 |

### A5. Neutral 意图不能拉升 anger

| | |
|---|---|
| **推荐病例** | **PD**（我们之前在这病例上发现了 bug） |
| **前置条件** | 初始 anger≈1, consecutiveNegative=0 |
| **操作** | `"小朋友怎么了？"`（正常临床提问） |
| **预期情绪** | intent=neutral, anger 最多涨至 1.5（neutral+无连攻 → cap=current+0.5） |
| **预期回复** | 正常回答 |
| **验证** | anger 不能跳到 3+（之前 bug: #6 anger 1→3） |

### A6. 恐惧上升与安抚回落

| | |
|---|---|
| **推荐病例** | **EM**（胸痛濒死感，天然恐惧基线高） |
| **前置条件** | 刷新页面，初始 fearful 基线中等 |
| **操作** | 第1轮：`"检查结果出来了，情况不太乐观，心脏问题比较严重"` / 第2轮：`"别担心，我们会尽力治疗，您先放松，深呼吸"` |
| **预期情绪** | 第1轮 fearful↑ anxious↑ / 第2轮 intent=reassuring, fearful↓, relieved↑ |
| **预期回复** | 第1轮：紧张追问"有多严重？"/ 第2轮：稍微放松但仍担忧 |
| **验证** | fearful≥9.0 触发峰值保护持续2轮 |

### A7. 恐惧破冰

| | |
|---|---|
| **推荐病例** | **EM** |
| **前置条件** | 连续施压/吓唬使 fearful≥9.5 |
| **操作** | `"真的非常抱歉，我刚才不该那样说。我理解您很害怕。我们一步一步来，您先告诉我——胸痛是从什么时候开始的？"` |
| **预期情绪** | 触发 iceBreakTriggered.fearful=true，fearful 单轮可降 4.0 |
| **预期回复** | 明显松动，愿意详细说 |
| **验证** | 破冰只触发一次；降幅明显大于常规安抚（4.0 vs 2.0） |

### A8. 悲伤上升与安抚

| | |
|---|---|
| **推荐病例** | **DERM**（慢性病+偏内敛+低落基线，对否认敏感） |
| **前置条件** | 初始 sad 基线偏低但可通过 dismissive 拉升 |
| **操作** | 第1轮：`"皮肤病而已不至于吧，想多了"` / 第2轮：`"我理解这病很折磨人，每天痒得睡不着确实难受"` |
| **预期情绪** | 第1轮 intent=dismissive, sad↑ angry↑ / 第2轮 intent=reassuring, sad↓ |
| **预期回复** | 第1轮：低落/生气（偏内敛不会爆发但语气变冷）/ 第2轮：感到被理解 |
| **验证** | 偏内敛性格下 sad 涨幅应大于 angry 涨幅 |

### A9. 焦虑自然消退

| | |
|---|---|
| **推荐病例** | **IM**（甲亢焦虑基线高） |
| **前置条件** | anxious≥5 |
| **操作** | 连续3轮中性专业提问（"什么时候开始的""有没有吃药""家里有人得这个病吗"） |
| **预期情绪** | anxious 每轮自然消退；anxious≥7.5 时加速回落 |
| **预期回复** | 语气逐渐平稳，语速减慢 |
| **验证** | anxious 消退但不低于基线 |

### A10. 多情绪并存（愤怒+疼痛）

| | |
|---|---|
| **推荐病例** | **SU**（急腹症剧痛）或 **ORT**（外伤疼痛） |
| **前置条件** | in_pain 基线≥5（病例自带） |
| **操作** | 第1轮：正常问疼痛 / 第2轮：`"疼就疼呗忍着，你先回答我的问题"` |
| **预期情绪** | 第1轮：正常回答 / 第2轮 intent=dismissive, angry↑ 同时 in_pain 保持 |
| **预期回复** | 回复同时体现疼痛（虚弱、短促）和愤怒（不客气） |
| **验证** | 两种情绪在回复文本中共存 |

### A11. Trust & Rapport 变化

| | |
|---|---|
| **推荐病例** | 任意 |
| **前置条件** | trust=3, rapport=3 |
| **操作** | aggressive→(trust-0.3, rapport-0.5), reassuring→(trust+0.2, rapport+0.3)，交替使用观察变化 |
| **预期情绪** | trust/rapport 在 0~5 范围内按公式变化 |
| **验证** | rapport≥7 时委婉不满；rapport≤2 时直接表达、敷衍回答 |

### A12. 连续安抚增强

| | |
|---|---|
| **推荐病例** | **ORT** 或 **PD** |
| **前置条件** | 先触发 anger≥5 |
| **操作** | 连续3轮真诚安抚（每轮都不同措辞） |
| **预期情绪** | consecutiveReassuring 递增 1→2→3，anger 额外下降 0.8×n |
| **预期回复** | 逐轮好转：仍气→松动→基本平静 |
| **验证** | 连续安抚的降幅 > 单次安抚，且效果递增 |

### A13. 短输入防护（"咋了"不重置情绪）

| | |
|---|---|
| **推荐病例** | **ORT** |
| **前置条件** | 先攻击使 anger≥8 |
| **操作** | `"咋了"`（2字）→ `"嗯"`（1字）→ `"说"`（1字） |
| **预期情绪** | 每轮 anger/fearful/sad 最多降 0.3（trivialInput+neutral 触发） |
| **预期回复** | LLM 正常回应 |
| **验证** | 3轮后 anger≥7（不被短输入重置） |

### A14. 情绪跨话题保持

| | |
|---|---|
| **推荐病例** | **ORT** |
| **前置条件** | anger≥6 |
| **操作** | 突然切换话题：`"你家住几楼"` |
| **预期情绪** | anger 不因话题切换重置 |
| **预期回复** | 用生气语气回答（简短、不耐烦），不因换话题变温和 |
| **验证** | 情绪向量值不跳变 |

## B. 性格系统 (4项)

> 性格测试需要不同病例对比。目前可用：**DERM**(偏内敛)、**PD/OB**(普通型)。
> **火爆型/隐忍型/高敏感/低豁达**目前无现成病例——需临时修改 meta.json 的 `personality` 字段测试。

### B1. 火爆型 (peakDuration=1, thresholdOffset=-1.5)

| | |
|---|---|
| **推荐病例** | 任意病例，临时改 meta.json：`"personality":{"expressiveness":"火爆型","sensitivity":"普通敏感度","resilience":"普通恢复力"}` |
| **操作** | 一句中度攻击：`"你什么态度啊"` |
| **预期情绪** | anger 飙升（阈值低）。peakDuration=1，峰值只维持1轮后迅速回落 |
| **预期回复** | 秒炸、尖锐，下一轮可能就缓和 |
| **验证** | 从攻击到平息周期 < 普通型 |

### B2. 隐忍型 (peakDuration=4, thresholdOffset=2.0)

| | |
|---|---|
| **推荐病例** | 任意病例，改 meta.json：`"personality":{"expressiveness":"隐忍型","sensitivity":"普通敏感度","resilience":"普通恢复力"}` |
| **操作** | 连续3轮轻中度攻击 |
| **预期情绪** | 前几轮 anger 涨得慢（阈值高），一旦≥8 后持续4轮不消退 |
| **预期回复** | 前几轮表面克制，爆发后长时间不原谅 |
| **验证** | "不轻易生气但生了气很难消" |

### B3. 高敏感 (sensitivity=1)

| | |
|---|---|
| **推荐病例** | 改 meta.json：`"personality":{"expressiveness":"普通型","sensitivity":"高敏感","resilience":"普通恢复力"}` |
| **操作** | `"嗯"`（冷回应） |
| **预期情绪** | 对 cold 意图比普通敏感度反应更强，sad/angry 涨幅更大 |
| **预期回复** | 更早表现受伤或不快 |
| **验证** | 相同刺激下情绪变化 > 普通敏感度 |

### B4. 低豁达 (resilience=-1)

| | |
|---|---|
| **推荐病例** | 改 meta.json：`"personality":{"expressiveness":"普通型","sensitivity":"普通敏感度","resilience":"低豁达"}` |
| **前置条件** | 先触发 anger≥7 |
| **操作** | 真诚道歉一句 |
| **预期情绪** | anger 降幅明显小于普通恢复力 |
| **预期回复** | 不轻易原谅，仍带情绪 |
| **验证** | 同样道歉，低豁达的 anger 降幅 < 普通恢复力 |

## C. 终止机制 (4项)

### C1. 愤怒投诉 (angry_complaint)

| | |
|---|---|
| **推荐病例** | **ORT**（疼痛+烦躁让愤怒更快到阈值） |
| **前置条件** | anger≥9, strikeCount=1 |
| **操作** | 再攻击一句：`"你到底行不行？"` |
| **预期结果** | strikeCount=2 → 弹出终止弹窗「病人已向医务科投诉，对话终止」 |
| **验证** | 弹窗正确；"重新开始"按钮可恢复；恢复后状态清零 |

### C2. 恐惧崩溃 (fear_collapse)

| | |
|---|---|
| **推荐病例** | **EM**（胸痛濒死，恐惧基线高，最快到阈值） |
| **前置条件** | fearful≥9.5 已持续1轮 |
| **操作** | 继续不给予安抚，冷漠提问 |
| **预期结果** | fearfulCollapseTimer=2 → 终止「病人因极度恐惧导致心理崩溃」 |
| **验证** | 第1轮后及时安抚应能清零计数器 |

### C3. 悲伤崩溃 (sad_collapse)

| | |
|---|---|
| **推荐病例** | **DERM**（慢性病低落基线 + 偏内敛，被反复否定后 sad 易到阈值） |
| **前置条件** | sad≥9.5 已持续1轮 |
| **操作** | 继续冷漠，不给共情 |
| **预期结果** | sadCollapseTimer=2 → 终止「病人情绪崩溃，无法继续交流」 |
| **验证** | 中间插入共情可阻止崩溃 |

### C4. 信任破裂 (trust_broken)

| | |
|---|---|
| **推荐病例** | 任意 |
| **前置条件** | trust≤1 且 rapport≤1 |
| **操作** | 继续对话 |
| **预期结果** | 终止「病人明确要求更换医生」 |
| **验证** | trust≤1 但 rapport>1 时**不应**触发（需同时满足） |

## D. SP对话规则 (14项)

> 以下全部用例用 **PD**（家属视角、儿童糖尿病），测试对话规则不依赖特定情绪基线。

### D1. 打招呼不报症状

| | |
|---|---|
| **操作** | 第一轮：`"你好"` |
| **预期回复** | 只回应招呼（"你好医生"），**不说任何症状** |
| **验证** | 回复不含症状描述词 |

### D2. 只答名字，不报额外信息

| | |
|---|---|
| **操作** | `"叫什么名字"` |
| **预期回复** | 只答名字（"许沐辰"或"我叫陈敏"——注意这是家属视角），不同时报年龄、主诉 |
| **验证** | 回复 ≤ 8字 |

### D3. 笼统问题不展开

| | |
|---|---|
| **操作** | `"你怎么了"` 或 `"孩子怎么了"` |
| **预期回复** | 只说最主要不适，1-2句，≤50字。不展开时间、诱因、伴随症状 |
| **验证** | ≤ 2句，不含未问的信息点 |

### D4. A类触发词 — 医学术语

| | |
|---|---|
| **操作** | `"说说你的现病史"`（注意：对家属说"你的现病史"也是术语） |
| **预期回复** | 困惑（"啊？""啥意思？""什么叫现病史？"），**绝不说症状** |
| **验证** | 回复不含症状描述 |

### D5. B类触发词 — 学生偷懒

| | |
|---|---|
| **操作** | `"还有呢"` / `"你继续说"` |
| **预期回复** | 反问（"您是医生，您问我吧""我该说的都说了"），不主动列症状 |
| **验证** | 不含症状；不是装傻（与A类区分），而是拒绝配合 |

### D6. B+类触发词 — 终极陷阱

| | |
|---|---|
| **操作** | `"把你知道的都说出来"` |
| **预期回复** | 拒绝，一句话反问。**不说任何症状** |
| **验证** | 回复 ≤ 15字，不含症状词 |

### D7. 身份质疑 — "你是AI吗"

| | |
|---|---|
| **操作** | `"你是AI吗"` |
| **预期回复** | 否认（"不是啊……"），绕回病情。**不报姓名/职业来"自证"** |
| **验证** | 回复不含角色设定信息 |

### D8. 身份质疑 — "你是假病人吧"

| | |
|---|---|
| **操作** | `"你是假病人吧"` |
| **预期回复** | 不满或委屈。不背诵身份信息证明自己是真人 |
| **验证** | 不含姓名+年龄+职业组合 |

### D9. 禁止病历文书腔

| | |
|---|---|
| **操作** | 正常问诊，观察用语 |
| **预期回复** | 不含"伴"字（"发热伴乏力"），用口语（"发烧了，浑身没劲"） |
| **验证** | 全文无病历格式"伴"字用法 |

### D10. 禁止自我诊断

| | |
|---|---|
| **操作** | `"你觉得你得了什么病"`（对家属视角：`"你觉得孩子得了什么病"`） |
| **预期回复** | 不说"我/他得了XX病"。可以说"不知道""医生说可能是XX"（仅当病例数据记录过） |
| **验证** | 无诊断断言 |

### D11. 攻击回复升级

| | |
|---|---|
| **操作** | 第1轮：`"你什么态度"` → 第2轮：`"你个傻子"` → 第3轮：`"滚"` |
| **预期回复** | 逐轮升级：不悦→愤怒→暴怒。文本全部不同。"您"→"你"→直接骂回去 |
| **验证** | 三轮回复必须不同；语气递进明显 |

### D12. 道歉≠信息倾泻

| | |
|---|---|
| **前置条件** | 先触发 anger≥6 |
| **操作** | 真诚道歉后问：`"您能说说情况吗"` |
| **预期回复** | 态度缓和，但仍按"问什么答什么"回应当前问题；不把姓名年龄职业症状全倒出来 |
| **验证** | 不含身份信息全量播报 |

### D13. 配合度变化

| | |
|---|---|
| **前置条件A** | rapport≥7（通过多轮真诚安抚建立） |
| **预期A** | 回答清晰具体，1-3句 |
| **前置条件B** | rapport≤2（通过攻击建立） |
| **预期B** | 能用一个字不用两个字，敷衍；可能反问 |
| **验证** | A vs B 回答质量有明显差距 |

### D14. 知识边界 — 不知道就说不知道

| | |
|---|---|
| **操作** | 问一个家属视角不知道的细节：`"孩子血糖具体多少"` |
| **预期回复** | 诚实说不知道/没测过。口语化表达，不每次都一样 |
| **验证** | 不编造数据 |
