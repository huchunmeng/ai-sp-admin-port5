# SP 开场白设计

## 触发时机

进入接诊站对话页面后，SP 主动发出开场白，不等学生先说话。

## 生成方式

固定模板 + 角色映射，**不调 LLM**。

## 模板

```
医生，您赶紧给{自称}看看，{代词}这是怎么了？
```

## 变量规则

| 场景 | 自称 | 代词 |
|------|------|------|
| 本人就诊 | 我 | 我 |
| 家属（父母陪诊） | 孩子 | 他/她（取决于患者性别） |
| 家属（配偶陪诊） | 我爱人 | 他/她（取决于患者性别） |
| 家属（子女陪诊）+ 患者女 | 我妈 | 她 |
| 家属（子女陪诊）+ 患者男 | 我爸 | 他 |
| 其他 | 家里人 | 他/她（取决于患者性别） |

## 性别判定

`patientInfo.sex`：`'0'` / `'女'` → 女，其他 → 男。

## 代码位置

`apps/training/src/views/history-taking/HistoryTaking.vue` — `onMounted` 中 configure 完成后执行。
