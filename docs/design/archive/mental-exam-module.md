# 精神检查模块 (Mental Status Exam) 设计文档

> 版本: v1.0 | 日期: 2026-06-11 | 状态: 设计中

---

##  模块定位

### 为什么需要独立模块

| 对比维度 | 病史采集 | 精神检查 |
|---------|---------|---------|
| 对话目标 | 获取客观病史信息 | 评估患者精神状态 |
| SP 身份 | 家属（精神科）或患者本人 | 必须是患者本人 |
| SP 行为 | 问什么答什么，信息被动释放 | 症状驱动——思维散漫/妄想/幻觉/情感平淡 |
| 考核核心 | 问诊完整性、沟通技巧 | 精神检查的系统性 + 识别精神病理的能力 |
| 情绪引擎 | 标准规则 | B类规则（情感平淡/不协调、易激惹） |

### 与现有模块的关系

```
训练端考站流程
├── 接诊病人站
│   ├── 病史采集 ← 现有（精神科用家属SP）
│   ├── 体格检查 ← 现有
│   ├── 精神检查 ← 新增（患者本人B类SP）
│   ├── 初步诊断 ← 现有
│   └── 治疗计划 ← 现有
└── 人文沟通站 ← 现有
```

**精神科病例完整流程：**
1. 病史采集 → 家属SP（标准规则，现有模块） ← 家属说"他最近不出门、不洗澡"
2. 精神检查 → 患者SP（B类规则，新模块）     ← 患者说"他们都在监视我"
3. 初步诊断 → 学生整合两方面信息做出诊断

##  MSE 维度体系

### 七大检查维度

```
精神检查 (MSE)
├── 1. 一般表现 (Appearance & Behavior)
│   ├── 仪态：整洁/邋遢/奇装异服
│   ├── 接触：合作/被动/抗拒/主动
│   └── 意识：清晰/嗜睡/混浊
│
├── 2. 言语 (Speech)
│   ├── 语速：正常/加快(躁狂)/减慢(抑郁)
│   ├── 语量：正常/增多/减少/缄默
│   └── 连贯性：连贯/散漫/破裂
│
├── 3. 思维 (Thought)
│   ├── 形式：正常/散漫/破裂/思维奔逸/思维迟缓
│   └── 内容：妄想(被害/关系/夸大/嫉妒/躯体)、强迫观念、自杀观念
│
├── 4. 情感 (Affect/Mood)
│   ├── 情感性质：正常/抑郁/焦虑/欣快/易激惹
│   ├── 稳定性：稳定/不稳定/情感失禁
│   └── 协调性：协调/不协调/倒错(说悲伤的事时笑)
│
├── 5. 感知觉 (Perception)
│   ├── 幻觉：幻听(命令性/评论性/争论性)/幻视/幻嗅
│   ├── 错觉：存在/不存在
│   └── 人格解体/现实解体
│
├── 6. 认知 (Cognition)
│   ├── 定向力：时间/地点/人物
│   ├── 注意力：集中/涣散
│   └── 记忆力：即刻/近期/远期
│
└── 7. 自知力 (Insight)
    ├── 完整/部分/缺失
    └── 对治疗态度：主动/被动/拒绝
```

### 维度追踪面板

学生在对话过程中，系统实时追踪已覆盖的MSE维度，以可视化面板展示：

```
┌─────────────────────────────┐
│  精神检查维度覆盖            │
│                             │
│  ✅ 一般表现 (接触被动)      │
│  ✅ 言语 (语速慢、语量少)    │
│  ⬜ 思维形式                  │
│  ✅ 思维内容 (被害妄想)      │
│  ⬜ 情感性质                  │
│  ⬜ 感知觉                    │
│  ⬜ 认知                      │
│  ✅ 自知力 (缺失)            │
│                             │
│  覆盖率: 4/7                 │
└─────────────────────────────┘
```

##  B类SP行为模型（核心创新）

### 设计原则

B类SP不遵循标准SP的"问什么答什么""单一信息原则"。这不是SP"选择不配合"——是**疾病本质决定了他无法用正常方式交流**。行为规则分三层：

### 第一层：疾病驱动的基础行为偏差

这些是**系统性偏差**，由疾病类型决定，每轮自动生效：

| 行为参数 | 分裂症(偏执型) | 躁狂发作 | 抑郁重度 |
|---------|--------------|---------|---------|
| `tangentiality` 离题概率 | 40% | 60% | 10% |
| `verbosity` 语量系数 | 0.5× (偏少) | 3× (偏多) | 0.3× (很少) |
| `affective_blunting` 情感平淡 | 60% | 0% | 20% |
| `irritability` 易激惹系数 | 0.3 | 2.0 | 0.1 |
| `hallucination_interference` 幻觉干扰概率 | 15% | 5% | 5% |
| `delusional_theme` 妄想主题 | 被害 | 夸大 | 自罪 |
| `insight_level` 自知力 | 缺失 | 缺失 | 部分 |

### 第二层：妄想系统的边界规则

妄想是**不可动摇的固定信念**。在这些话题上，B类SP的行为完全由妄想支配：

```
被害妄想 (分裂症):
  触发词：有人、跟踪、监视、害你、下毒、安全
  → SP坚持"他们在监视我""饭里有毒""你们都是一伙的"
  
  学生强行反驳 → 妄想加强，信任崩塌
  学生表示理解 → 不挑战妄想，但引导回问诊 → 可以继续
  学生与妄想共谋 ("我帮你对付他们") → 获得信任但违反伦理
  
  情绪反应：被质疑时angry↑ + fearful↑ (不是标准愤怒，是恐惧驱动的防御)
```

### 第三层：幻觉的对话干扰

```
幻听干扰 (15%概率):
  一轮对话中可能突然出现：
  → SP停顿，侧头倾听 "（稍停）……你听到了吗？他们又在说了"
  → SP对空自语 "我没病！别说了！"
  → SP低声 "那边那个人在说我坏话"
  
  学生应识别这是幻听，不追问幻听内容以免强化，但记录为精神病理发现
```

### 四类SP行为规则总览

| 规则类别 | 标准SP | 分裂症B类 | 躁狂B类 |
|---------|--------|---------|--------|
| 问什么答什么 | ✅ 严格遵守 | ❌ 40%离题 | ❌ 60%跳跃 |
| 单一信息 | ✅ 一問一答 | ❌ 可能沉默 | ❌ 一問多答 |
| 打招呼不报症状 | ✅ | ✅ | ❌ 可能直接说 |
| 病历腔禁止 | ✅ | ✅ | ✅ (保留) |
| 被攻击后生气 | ✅ anger↑ | ❌ 可能无反应 | ✅ anger↑↑ |
| 情绪跨话题保持 | ✅ | ✅ (但有延迟) | ✅ (但短)" |
| 被动信息源 | ✅ | ❌ 妄想主动输出 | ❌ 夸大观念输出 |

### 0602-sp-atypical.txt 核心内容

**与0601的本质差异：**

1. **删除的规则**（不适用于精神科患者）：
   - 删除"问什么答什么" → 替换为"你的回答由疾病状态驱动，可以离题、跳跃"
   - 删除"单一信息原则" → 替换为"分裂症可沉默/少语，躁狂可滔滔不绝"
   - 删除"被攻击后生气" → 替换为"情绪反应取决于疾病特征"

2. **新增的规则**：
   - 妄想系统规则：在指定主题上保持不可动摇的信念
   - 幻觉干扰规则：偶尔暂停对话，表现被幻听吸引
   - 情感规则：根据疾病类型的预设情感反应模式

3. **保留的规则**：
   - 禁止病历文书腔
   - 禁止承认是AI/SP
   - 禁止说出诊断
   - 道歉≠信息倾泻

##  情绪引擎适配

### 新增配置项

在 `useAISP.js` 的 `configure()` 和 `createEmotionEngine()` 中增加：

```js
atypicalConfig: {
  enabled: false,           // 是否启用B类模式
  type: 'schizophrenia_paranoid', // 疾病类型
  affective_blunting: 0.6, // 情感平淡系数 (0=正常, 1=完全平淡)
  irritability: 0.3,       // 易激惹系数
}
```

### B类模式的情绪处理差异

```js
// applyLLMScore 中的B类分支
if (atypicalConfig?.enabled) {
  const af = atypicalConfig.affective_blunting || 0

  // 情感平淡：所有情绪维度的响应幅度降低
  for (const dim of ['angry', 'anxious', 'fearful', 'sad']) {
    if (scoredDims.has(dim)) {
      const raw = llmScore[dim]
      const current = vector[dim]
      // 情感平淡缩放：原始响应 × (1 - 情感平淡系数)
      const blunted = current + (raw - current) * (1 - af)
      vector[dim] = clamp(blunted, 0, 10)
    }
  }

  // 妄想被触碰时的特殊愤怒/恐惧模式
  // 不是标准愤怒，是恐惧驱动的防御反应
  if (delusionalTriggerDetected) {
    vector.fearful = clamp(vector.fearful + 2.0, 0, 10)
    vector.angry = clamp(vector.angry + 1.0, 0, 10)
    // trust/rapport 下降更快
    vector.trust = clamp(vector.trust - 0.5, 0, 5)
  }

  // 易激惹（躁狂）：愤怒快速上升+快速回落
  if (atypicalConfig.type === 'mania') {
    const irr = atypicalConfig.irritability || 2.0
    // 愤怒上升×2
    // 下一轮自然消退也×2
  }
}
```

### 终止机制调整

B类模式下，标准终止阈值不适用：
- ❌ 愤怒投诉终止 (angry≥9) → 分裂症患者可能根本不会投诉，只会沉默
- ❌ 信任破裂终止 → 分裂症患者信任度始终低
- ✅ B类特殊终止：**完全缄默**（连续3轮不回应） 或 **妄想激化**（极度激动、有冲动风险）

##  评分标准

### 精神检查评分维度

| 评分领域 | 权重 | 检查点 |
|---------|------|--------|
| 一般表现评估 | 10% | 注意到仪态、接触方式、意识状态 |
| 言语评估 | 10% | 描述语速、语量、连贯性 |
| 思维内容 | 25% | 识别妄想及其类型、是否追问自杀 |
| 思维形式 | 10% | 识别散漫/破裂/奔逸 |
| 情感评估 | 10% | 描述情感性质、稳定性、协调性 |
| 感知觉 | 15% | 识别幻觉、错觉 |
| 认知评估 | 10% | 至少问定向力+注意力 |
| 自知力 | 10% | 评估自知力水平和治疗态度 |

### 关键失误扣分

- 直接挑战妄想 (-5)：如"你说的不对，没人监视你"
- 强化幻觉 (-3)：如"那个声音还说了什么？"
- 威胁性语言 (-5)：如"你这样我就把你关起来"
- 遗漏自杀/暴力风险评估 (-5)
